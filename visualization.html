Phase 1 · 内容采集

  • B站链接 → 提取 BV 号
  • Playwright 获取新鲜 cookies
  • B站 API → 获取视频流地址
  • qn=80 1080P 高清下载

🎙️

Phase 2 · 语音转写

  • faster-whisper 引擎
  • 支持中文 -l zh
  • 支持英文 -l en / 俄语 -l ru
  • 模型选择: base / small / large-v3
  • 输出: 时间轴文本 + SRT 字幕
  • HF 国内镜像 hf-mirror.com
🧠

Phase 3 · LLM 结构化

  • 阅读 Whisper 全文转写
  • LLM 提炼核心观点 + 关键洞察
  • 自动生成 YAML front-matter
  • 判断内容分类
  • 生成 [[双向链接]] 关联笔记
📓

Phase 4 · Obsidian 入库

  • 按分类存入 Notes/ 目录
  • AI学习 / 编程开发 / 经济学 / 语言 / 学术 / 日常
  • 标准 YAML front-matter 格式
  • 自动创建分类文件夹
  • 兼容 Obsidian Graph View
🗺️

Phase 5 · XMind 联动

  • 纯 Python 零依赖解析 XMind
  • 支持 XMind 8 + XMind 2020+
  • 每个节点 → 独立 .md 笔记
  • 自动建立 [[双向链接]]
  • 自动生成 MOC 索引页
  • Obsidian Graph 可视化

🧠 设计理念
🪺

为什么叫 Knowledge Nest？

像鸟巢一样 —— 把零散的知识碎片（视频、文章、导图）衔回来，
整理成温暖、有序的「知识巢穴」。

四大原则

  • 零依赖优先 — XMind 解析器纯 Python 标准库实现
  • AI 增强，不替代 — Whisper + LLM，知识组织由你掌控
  • Obsidian 原生 — [[双向链接]]、Graph View、Tag 全兼容
  • 反爬友好 — B站下载绕过 412，Playwright 获取合法 cookies
