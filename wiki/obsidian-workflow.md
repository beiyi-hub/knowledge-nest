# Obsidian 工作流

## 推荐配置

### 核心插件

打开 `设置 → 核心插件`，开启：
- ✅ **反向链接** — 显示哪些笔记链接到当前页
- ✅ **出链** — 自动检测未创建的 `[[链接]]`
- ✅ **大纲** — 基于标题的目录导航
- ✅ **标签面板** — 按标签浏览笔记
- ✅ **图谱视图** — 可视化知识网络

### 社区插件推荐

- **Dataview** — 用 SQL 类似语法查询笔记库
- **Templater** — 高级模板引擎
- **Quick Add** — 快速捕获灵感

## 从视频笔记到知识网络

1. 看视频 → AI 生成笔记（含 [[双向链接]]）
2. N 个视频后 → Obsidian Graph View 自动形成网络
3. 发现「知识孤岛」→ 手动补充链接
4. 定期导出关键主题为 XMind 思维导图
5. XMind → 再导入 Obsidian，形成闭环

```

├── Notes/
│   ├── AI学习/         ← 机器学习、LLM

│   ├── 语言学习/        ← 雅思、俄语
│   ├── 学术研究/        ← 论文、研究方法
│   ├── 日常杂谈/        ← 时事、Vlog
│   └── 未分类/          ← 待整理的

├── Attachments/        ← 图片等附件
└── Projects/           ← 项目笔记
```
